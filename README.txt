GPT Assistant Telegram Bot

Описание:
Бот принимает сообщения в Telegram и пересылает их в OpenAI Assistants API. Затем ответы возвращаются пользователю в Telegram.

Подготовка:
1. Убедитесь, что в системе установлен Python 3.  
   Проверьте командой:
   python3 --version  
   Если Python 3 нет, установите:
   sudo apt update  
   sudo apt install -y python3 python3-venv python3-pip  

2. Склонируйте или скопируйте проект в удобную папку и перейдите в нее:
   cd GPT_ASSistant_Telegram_bot

3. Проверьте файл config.json:
   {
       "TELEGRAM_TOKEN": "<токен_бота>",
       "OPENAI_API_KEY": "sk-proj-...",
       "ASSISTANT_ID": "asst_..."
   }
   - TELEGRAM_TOKEN: токен вашего бота (полученный в BotFather)
   - OPENAI_API_KEY: ключ OpenAI (формат sk-proj-...)
   - ASSISTANT_ID: ID ассистента из OpenAI (формат asst_...)

Запуск:
1. Дайте право на выполнение скрипта:
   chmod +x run.sh

2. Запустите скрипт:
   ./run.sh
   Если данные в config.json не заполнены, скрипт спросит их в терминале.  
   После установки необходимых пакетов бот запустится автоматически.

Проверка работы бота:
1. В Telegram найдите своего бота по токену (через поиск по username).  
2. Отправьте ему любое сообщение или команду /start.  
3. Убедитесь, что бот отвечает. Если нет, перепроверьте ключи в config.json и запустите ./run.sh заново.

Типичные проблемы:
- Отсутствуют Python или модули: установите пакеты по инструкциям выше.  
- Ошибка токена: исправьте значения в config.json и перезапустите бот.  
- Ошибка Command not found: проверьте, что находитесь в папке бота и run.sh имеет право на выполнение (chmod +x run.sh).
